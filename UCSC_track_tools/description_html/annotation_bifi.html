Description

This track shows the annotation.

The Id of each track can be interpreted by splitting the string on each pipe symbol "|". The fields are:

  1. Annotation type
  2. Conitg/ Plasmid Id
  3. Feature type
  4. Annotation Id

Display Conventions and Configuration

The colours of the annotation corresponds with the reading frame of the gene:

  •  Red  + strand 1 frame
  •  Blue  + strand 2 frame
  •  Green  + strand 3 frame
  •  Purple  - strand 4 frame
  •  Orange  - strand 5 frame
  •  Turquoise  - strand 6 frame
