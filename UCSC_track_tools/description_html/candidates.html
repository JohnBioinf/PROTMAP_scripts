Description

This track shows a list of not annotated ORFs which have more than 6 good (FDR < 1).

These ORFs are considered as novel protein candidates. A full list of all candidates with further information can be found as supplement material of the publication "Direct mapping of peptide-spectral-matches to genome information facilitates qualifying proteomics information" here.

Display Conventions and Configuration

The colours of the annotation corresponds with the reading frame of the gene:

  •  Red  + strand 1 frame
  •  Blue  + strand 2 frame
  •  Green  + strand 3 frame
  •  Purple  - strand 4 frame
  •  Orange  - strand 5 frame
  •  Turquoise  - strand 6 frame
